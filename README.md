# Real-time Chat App (Socket.IO)

A minimal real-time chat application built with **Node.js**, **Express**, and **Socket.IO**.

## Features

- Real-time messaging between connected clients
- Join/leave system notifications
- Timestamped chat messages
- Simple responsive UI

## Run locally

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the server:

   ```bash
   npm start
   ```

3. Open the app in your browser:

   ```
   http://localhost:3000
   ```

4. Open multiple tabs/windows to test real-time messaging.
